Arrays in JS

In JavaScript, an array is a data structure that allows you to store manipulate a collection of values under a single variable. Arrays can hold elements of different data types, including numbers, strings, objects, and even other arrays. Let's explore the key concepts of arrays in JavaScript.

Creating and Accessing Arrays

You can create an array by enclosing its elements in square brackets []. Here's an example of creating an array with numbers: let numbers = [1, 2, 3, 4, 5]; Elements in an rray are accessed using their index, starting from 0. For example, to access the first element in the array: console.log(numbers[0]); // Output: 1 To access the last element in the array: console.log(numbers[numbers.length - 1]); // Output: 5

Array Methods

JavaScript arrays provide several built-in methods to manipulate and perform operations on the elements. Some Commonly used array methods include: push, pop, shift, unshift, splice, slice, concat, indexOf, includes, forEach, map, filter, and reduce. These methods allow you to add or remove elements, search for elements, iterate over the array, transform the array, and more.

Iterating Over Arrays

There are multiple ways to iterate over the elements of an array in JavaScript. You can use a traditional for loop with an index, or utilize array methods like forEach, map, filter, and reduce. These methods provide a concise and expressive way to iterate over arrays and perform operations on each element.

Multidimensional Arrays

JavaScript allows you to create multidimensional arrays, which are arrays of arrays. This means you can have arrays nested within other arrays, forming a matrix-like structure. Multidimensional arrays are useful when dealing with tabular data or nested structures. For example: let matrix = [ [1, 2, 3], [4, 5, 6], [7, 8, 9] ]; To access elements in a multidimensional array, you need to provide both the row and column indices. For example, to access the element at the second row and third column: console.log(matrix[1][2]); // Output: 6
